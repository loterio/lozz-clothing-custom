# Custom CSS - Lozz Clothing

Este projeto contém o *Código CSS Personalizado* desenvolvido para a loja **Lozz Clothing**, aplicado sobre um **tema padrão e gratuito** da plataforma **Nuvemshop**.

---

## Soluções Estratégicas

O foco deste ajuste foi elevar a percepção de marca e a usabilidade, utilizando técnicas modernas de *front-end* diretamente no painel de edição avançada.

* **Alinhamento Global (Sincronização):** Padronização de todas as seções (Vitrinas, Banners e Instafeed) para uma largura máxima de **1200px**, eliminando o aspecto desalinhado do tema nativo.
* **Design de Bordas e Sombras:** Aplicação de *border-radius* de **15px** e *box-shadow* suave para criar uma estética de "cards" moderna e sofisticada.
* **Correção de Sangria (Full-Bleed):** Ajuste de espaçamentos internos para que as imagens ocupem toda a extremidade dos banners, garantindo um acabamento limpo.
* **Layout Dinâmico e Flexível:** Implementação de *CSS Grid* e *Flexbox* para reorganizar elementos sem necessidade de alterar o código-fonte (HTML).
* **Responsividade Sob Medida:**
    * *Mobile:* Banners configurados em **1 coluna** para legibilidade total dos textos.
    * *Mobile:* Feed do Instagram ajustado para **2 colunas**, otimizando o aproveitamento de tela em smartphones.
